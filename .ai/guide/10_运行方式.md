# 运行方式

## Description

说明当前项目如何定位本机 Python 解释器。

## Trigger

以下情况优先参考本文档：

- 运行 Python 命令之前
- 做编译检查之前
- 做导入检查之前
- 运行测试之前
- 运行项目入口之前

## Instructions

- 当前项目不在文档里写死某一台机器的 Python 绝对路径
- 本机解释器路径放在：
  - `.ai/local/python_path.txt`
- 文件内容只写一行：
  - Python 解释器绝对路径
- 该文件只供本机使用，不提交到 git
- 运行脚本时，优先复用这个文件里的路径

### 运行测试时的临时目录

- 默认 pytest 会使用系统临时目录；在部分 Windows 环境下可能对 `%TEMP%` 无权限，导致 `tmp_path` 等 fixture 报错。
- 本项目约定：在**仓库根目录**执行测试时，将 pytest 的 basetemp 指到**本仓库**下的 `.pytest_tmp/`（已列入 `.gitignore`，勿提交）。
- 示例（PowerShell，先 `cd` 到项目根）：

```powershell
$py = (Get-Content .\.ai\local\python_path.txt -Encoding UTF8).Trim()
& $py -m pytest tests/unit tests/integration -q --basetemp .pytest_tmp
```

- 仅跑云端用例时同样可带上 `--basetemp .pytest_tmp`。
