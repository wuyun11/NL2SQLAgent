# 运行方式

## Description

说明当前项目如何定位本机 Python 解释器。

## Trigger

以下情况优先参考本文档：

- 运行 Python 命令之前
- 做编译检查之前
- 做导入检查之前
- 运行测试之前
- 运行项目入口之前

## Instructions

- 当前项目不在文档里写死某一台机器的 Python 绝对路径
- 本机解释器路径放在：
  - `.ai/local/python_path.txt`
- 文件内容只写一行：
  - Python 解释器绝对路径
- 该文件只供本机使用，不提交到 git
- 运行脚本时，优先复用这个文件里的路径
